    SSHの設定方法を忘れるので手順をメモしておきます。

    手順

    # 公開鍵認証用の鍵を生成
$ssh-keygen -t rsa -b 4096 -f ~/.ssh/dosan_startup
# 公開鍵をサーバーに送信
$ssh-copy-id -i ~/.ssh/dosan_startup.pub <user>@<IP address>
# configファイルを編集
$nano .ssh/config

    configファイルには以下のことを書く。

    Host ringo
  Hostname <IP address>
  User <user>
  Port <Port番号>
  IdentityFile <公開鍵認証用に生成した鍵>

例：
Host ringo
  Hostname 255.255.255.255
  User hogehoge
  Port 22
  IdentityFile ~/.ssh/dosan_startup

    最後に、以下のことを実行する。

    $eval `ssh-agent`
$ssh-add -k ~/.ssh/dosan_startup
