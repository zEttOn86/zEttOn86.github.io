    はじめに

    去年に引き続いて2025年の振り返りと2026年について考えたいと思います。
    今年も子どもが寝ている時間に書きます。

    • 2024年の振り返りと2025年について

    よかったこと

    去年の一番よかったことは子どもが無事に生まれたことでしょうか。
    色んな方に協力いただきながら、無事にお迎えできました。
    仕事のほうも調整していただき、とても恵まれた環境だったと感じました。
    他には生活リズムを変えました。子どもの生活時間に合わせて9時就寝、4時起床に変えました。
    俳優の田中麗奈さんの雑誌記事を見て真似をしてみました。
    これが割とよかったです。今年も続けていきたいです。

    書いた記事

    去年は6本の記事を書いたようです。大体、読書の記録でした。技術書としてはUI系の本を中心に読みました。
    「UXデザインの法則 最高のプロダクトサービスを支える心理学」が一番印象に残っています。
    特に「美的ユーザビリティ効果」1を知れたのがよかったです。美しいUIの場合、多少、不便でもユーザは不満を持ちにくいという心理学です。
    美しいUIはやはり重要だということを改めて認識させられました。

    1. Gitでコミット済みの内容に修正を加える方法
    2. 「ペッパーズ・ゴースト」を読んだ
    3. 「UIデザイン　必携 - ユーザーインターフェースの設計と改善を成功させるために」を読んだ
    4. 「UIデザイン　みんなで考え、カイゼンする」を読んだ
    5. 「UXデザインの法則 最高のプロダクトサービスを支える心理学」を読んだ
    6. How to Disassemble TOSHIBA Satelite T652

    触った言語・フレームワーク・ツール

    以下の言語を触りました。

    • Java
    • C
    • Python

    一番触ったのはJavaです。印象に残った処理としては、パイプライン処理です。
    どのようにスレッドを立てて実行するかを工夫した気がします。
    振り返りとしては、マルチスレッド・プロセスのデザインパターンをよく知らないので勉強していきたいと感じました。


    C/Pythonは惰性ですが触っています。
    大きなことはあまりしなかった気がします。

    ツールはChatGPTを主に使いました。課金してほぼ毎日使っています。主に英語の勉強に使っています。
